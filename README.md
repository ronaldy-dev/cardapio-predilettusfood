# 🍔 Predilettu's Food - Catálogo Digital

Este é um projeto de catálogo digital mobile-first desenvolvido para otimizar o fluxo de pedidos de estabelecimentos de alimentação (Hamburguerias, Pizzarias e Sorveterias). O sistema permite que o cliente navegue pelo cardápio, adicione itens ao carrinho e finalize o pedido diretamente via WhatsApp.

## 🚀 Demonstração
O projeto está publicado e pode ser acessado aqui: 
[project-j9ma1.vercel.app](https://project-j9ma1.vercel.app)

## ✨ Funcionalidades
- **Interface Mobile-First:** Design otimizado para smartphones.
- **Carrinho Dinâmico:** Soma automática de valores e contagem de itens.
- **Filtro por Categorias:** Navegação rápida entre Hambúrgueres, Pizzas e Sobremesas.
- **Integração com WhatsApp:** Gera uma mensagem formatada com o resumo do pedido e total para o estabelecimento.
- **Dark Mode Premium:** Identidade visual inspirada na marca Predilettu's.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização com Flexbox, Grid e variáveis CSS para fácil manutenção.
- **JavaScript (Vanilla):** Lógica de manipulação do DOM e integração com API do WhatsApp.
- **Vercel:** Hospedagem e Deploy Contínuo.
